



IAsyncDatapin
=============

.. py:class:: IAsyncDatapin
   :canonical: ansys.engineeringworkflow.api.iasyncworkflow.IAsyncDatapin



Bases: :py:obj:`IAsyncElement`, :py:obj:`abc.ABC`


   Provides a runtime placeholder for some value of a particular type.

   The placeholder changes as the workflow runs. It can be linked to other datapins via direct
   links or equations.
