Property detail#

property IElement.element_id: str#
Abstractmethod:

Unique ID for the element.

This ID is assigned by the system.

property IElement.parent_element_id: str#
Abstractmethod:

Parent element’s ID.

If this is the root element of the workflow, the parent ID is a blank string.

property IElement.name: str#
Abstractmethod:

Name of the element.

property IElement.full_name: str#
Abstractmethod:

Full name of the element.

The full name is returned in dotted notation, starting from the root of the workflow.
